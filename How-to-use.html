Before you begin

System requirements

Chronos is lightweight and designed to run on virtually any modern Windows PC. Check these before starting.

🪟

Windows 10 or 11

Both 32-bit and 64-bit editions are supported. Windows 7/8 are not.

🎤

Microphone

Any USB, built-in, or Bluetooth mic. Set it as the default recording device in Windows Sound Settings.

🔊

Audio output

Speakers or headphones required for Chronos' TTS voice responses.

💾

~200 MB disk space

Includes the Vosk speech model and TTS cache storage. The cache grows slowly over time.

🗜️

ZIP extractor

Windows' built-in tool or WinRAR both work. WinRAR gives a handy right-click "Extract Here" shortcut.

⚡

No internet required

After the initial download, Chronos runs entirely offline. No Wi-Fi, no cloud, no accounts.

Step 01

Download the ZIP file

Grab the latest official Chronos release from the download page.

01

Get the official download

Download

Head to the Download page of this website and click the "Download Chronos.zip" button. The file is a single self-contained ZIP archive — everything Chronos needs to run is bundled inside.

📥
Always download Chronos from this official website. Third-party mirrors or file-sharing sites may serve outdated or tampered builds. The official download link is on the Download page.
⚠️
Browser warning: Your browser may show a warning like "This file type may be harmful." This is a standard warning for any executable inside a ZIP. Click Keep or Download anyway to proceed — Chronos contains no malware.

Once the download finishes, you should see Chronos.zip in your Downloads folder. Do not attempt to open or run files directly from inside the ZIP — always extract first.

Step 02

Extract the archive

Unzip Chronos.zip using Windows' built-in extractor or WinRAR.

02

Using Windows' built-in tool

Extract

Right-click Chronos.zip in File Explorer, then click "Extract All…". A dialog will appear asking where to extract. Choose a location and click Extract.

📂
Windows will create a new folder called Chronos at your chosen location containing all the extracted files. This is normal and expected.
02

Using WinRAR (recommended)

Extract

WinRAR adds a convenient "Extract Here" option directly in your right-click context menu, making extraction one click. It also supports more archive formats and gives progress feedback for large files.

🗜️
Don't have WinRAR? Download it free from the official site: rarlab.com/download.htm. Choose the version matching your Windows — 64-bit is correct for most modern PCs. WinRAR is free to use indefinitely.

Once WinRAR is installed, right-click Chronos.zip → "Extract Here" and the Chronos folder appears instantly in the same location as the ZIP.

Step 03

Place Chronos in a safe folder

Move the extracted folder to a permanent home — somewhere it won't be accidentally deleted or moved.

03

Choose a permanent location

Place

Cut and paste the Chronos folder from your Downloads into one of these recommended locations. Chronos stores its TTS cache relative to its own folder, so moving it later is fine as long as you also move the whole folder together.

C:\Program Files\Chronos
C:\Users\YourName\Documents\Chronos
C:\Users\YourName\AppData\Local\Chronos
D:\Apps\Chronos  (any drive works)
⚠️
Avoid leaving it in Downloads or Temp. These folders are frequently auto-purged by disk cleanup tools and Windows Storage Sense. If Chronos disappears, it's usually because of this. Move it somewhere stable.
💡
Program Files tip: If you install to C:\Program Files\Chronos, you may need to right-click the folder and set write permissions for your user account so Chronos can write to its TTS cache. Alternatively, just use your Documents or AppData folder to avoid this entirely.
Step 04

Launch Chronos

Double-click Chronos.exe to fire it up for the first time.

04

Running Chronos.exe

Launch

Open the Chronos folder and double-click Chronos.exe. On first launch it may take a few seconds as Chronos initialises the Vosk speech model and sets up its TTS cache directory.

🛡️
Windows SmartScreen warning: You may see a blue dialog reading "Windows protected your PC." This appears because Chronos is not code-signed with an expensive certificate — it does not mean it's harmful. Click "More info" → "Run anyway" to proceed. You'll only see this once.
🎤
Microphone permission: Windows may also ask "Do you want to allow Chronos to access your microphone?" Click Yes — this is essential for voice recognition to work. You can manage this later in Settings → Privacy → Microphone.
📌
To launch Chronos quickly in future: right-click Chronos.exe → "Send to" → "Desktop (create shortcut)". You can also drag the shortcut to your taskbar to pin it.
04b

What happens on startup

Info

When Chronos starts successfully, it will:

✓

Greet you with the date and time

Chronos speaks a welcome message including today's date and current time using its natural TTS voice.

✓

Begin listening immediately

The microphone is activated and Vosk starts processing audio. No wake word is needed by default — Chronos is always ready.

✓

Load the TTS cache

Any previously cached audio is loaded for instant playback. On first run the cache is empty, so the first responses will generate audio fresh — which takes a moment.

Step 05

Start talking to Chronos

Just speak naturally — Chronos listens, understands, and responds.

05

Speaking clearly

Use it

Chronos uses the Vosk offline speech recognition engine. Vosk is trained on natural speech but performs best when you speak clearly and at a moderate pace. You do not need to yell or over-enunciate — just talk the way you normally would.

💡
Background noise: Vosk handles mild ambient noise well, but loud music, TV in the background, or echo-heavy rooms can reduce accuracy. A headset microphone will give the best results.

Chronos processes your audio entirely on-device using your CPU. There is no internet round-trip, which means recognition is fast regardless of your connection speed.

05

How the TTS cache works

Performance

Every time Chronos speaks a phrase using edge-tts, the generated audio is saved permanently in a local cache folder. The next time Chronos needs to say the same phrase, it plays the cached file instantly — no generation delay.

⚡
The TTS cache is never cleared — it grows over time and Chronos only gets faster with repeated use. Greetings, common responses, and frequently used phrases will always snap back instantly after the first time they're generated.

This is especially noticeable with the daily greeting: the first time Chronos greets you on a new date, it generates fresh audio. From that point on, it plays back in milliseconds.

Tips & best practices

Getting the most out of Chronos

Small habits that make a big difference in day-to-day use.

✓

Use a headset or close-field mic

Vosk accuracy improves significantly with a microphone closer to your mouth. Even a budget USB headset beats a far-field laptop mic in a noisy room.

✓

Set your mic as the Windows default

Go to Settings → System → Sound → Input and confirm that your intended microphone is selected as the default device. Chronos uses whatever Windows reports as default.

✓

Keep Chronos in one permanent folder

The TTS cache lives inside the Chronos folder. Moving just the .exe will break the cache lookup. Always move the entire Chronos folder together if you relocate it.

✓

Let it finish speaking before issuing the next command

Chronos is sequential — it processes one command at a time. Interrupting mid-speech may cause the next command to be missed. Wait for the response to finish.

✓

Check the console window for feedback

Chronos opens a console/terminal window alongside its main process. If it's not responding to voice, the console will usually show the reason — such as "no audio input detected" or a recognition error.

✓

Create a desktop shortcut for easy access

Right-click Chronos.exe → Send to → Desktop (create shortcut). Pin the shortcut to your taskbar so Chronos is always one click away.

✓

Use the "reset" voice command if things get sluggish

Saying "reset" or "clear" performs a soft purge of Chronos' short-term memory state without closing the app or touching the TTS cache. Use this if responses start feeling out of sync.

Troubleshooting

Common issues & fixes

Most problems have a simple cause. Check these first before anything else.

!

Chronos doesn't hear me / no response to voice

This is almost always a microphone configuration issue, not a bug.

🔧
1. Open Windows Settings → System → Sound → Input and confirm your mic is selected as the default device.

2. Check that the microphone volume level is above 50%.

3. Confirm that Chronos has microphone permission: Settings → Privacy → Microphone → ensure it's toggled on for desktop apps.

4. Check the Chronos console window — it will show whether audio is being received.
!

Chronos crashes on startup

A crash on launch is usually caused by a missing file inside the Chronos folder, or Chronos being run directly from inside the ZIP without extracting first.

⚠️
Never double-click Chronos.exe from inside the ZIP. Always extract the entire ZIP first, then run the .exe from the extracted folder. Running from inside a ZIP causes files to be missing at runtime.

If you extracted properly and still see a crash, re-download the ZIP from the Download page and extract fresh. The download may have been corrupted mid-transfer.

!

Voice recognition is inaccurate

Vosk's accuracy depends on audio quality. Try these in order:

💡
• Move closer to your microphone.
• Reduce background noise (TV, music, fans).
• Speak at a steady moderate pace — not too fast, not exaggerated.
• Switch to a headset microphone if using a laptop's built-in mic.
• Check that the correct input device is selected in Windows Sound settings.
!

No sound / Chronos speaks but I can't hear it

Check that your speakers or headphones are connected and selected as the default playback device in Windows Sound settings. Also confirm your system volume is not muted. Chronos plays audio through the Windows default output device — changing that in the Sound control panel is sufficient.

!

Antivirus flagging Chronos

Some antivirus engines flag unsigned executables as potentially unwanted programs (PUPs). Chronos is not malware — it is a locally built, unsigned application with no code-signing certificate. You can add the Chronos folder as an exclusion in your antivirus software to prevent interference.

ℹ️
If you're unsure, you can verify the download by comparing the SHA-256 hash of your downloaded ZIP against the hash published on the Download page.

Uninstalling

Removing Chronos

Chronos leaves no footprint on your system outside its own folder.

⚠️ To completely remove Chronos

Simply delete the Chronos folder. Chronos does not write to the Windows registry, does not install services or drivers, and does not place files anywhere outside its own folder. There is no uninstaller because none is needed — deleting the folder is the complete and total uninstallation. You can also remove the desktop shortcut if you created one.

ℹ️
If you want to move Chronos rather than remove it, simply move the entire folder to a new location and re-create your desktop shortcut pointing to the new path. The TTS cache inside the folder moves with it automatically.
