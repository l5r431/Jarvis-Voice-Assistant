Zara Assistant (“we”, “our”, or “the Assistant”) is a desktop application designed to help users perform tasks such as opening applications, executing system commands, and automating workflows.

1. Information We Collect

Voice Input

When voice features are enabled, the Assistant processes microphone input to detect commands. Audio is not permanently stored unless explicitly configured.

Commands & Usage

We may process commands such as “open Chrome” or “launch VS Code” to execute actions and improve performance.

Device Info

The Assistant may access system-level details like OS version and installed applications strictly for functionality.

2. How We Use Data

  • Execute commands
  • Improve performance
  • Fix errors and bugs
  • Customize user experience

3. Local vs Cloud Processing

Local Mode

All data is processed on your device. No internet required.

Cloud Mode (Optional)

Some features may use external APIs. Data is securely transmitted and minimized.

4. Data Storage

Most data is stored locally. Temporary data is deleted after use. Logs are optional and remain on your device.

5. Permissions

  • Microphone (voice commands)
  • File system access (launch apps)
  • Internet (optional cloud features)

6. Security

We implement encryption and secure practices, but no system is completely secure.

7. User Control

You can disable features, clear data, or uninstall the Assistant at any time.

8. Changes to Policy

We may update this policy. Continued use means acceptance of updates.
