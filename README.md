## **Chronos** 

Chronos is a powerful desktop voice assistant for Windows designed to improve productivity through fast voice-controlled automation. It combines intelligent voice commands with a clean graphical interface, allowing users to control applications, media, system functions, and text input hands-free. 

## **Features** 

## **Voice-Controlled Application Management** 

- Open desktop applications instantly 

- Close applications by: 

- focused window 

- application name 

- Minimize and maximize windows 

- Detect and list currently open applications 

## **System Controls** 

- Adjust system volume 

- Mute and unmute audio 

- Restart the PC 

- Shut down the PC 

## **Media Playback Controls** 

- Play and pause media 

- Skip to next track 

- Return to previous track 

## **Website Launching** 

- Open websites directly using voice commands 

## **Dictation Mode** 

Chronos includes a built-in dictation/input mode that allows users to type using their voice. 

Activate dictation mode by saying: 

- `input` 

- • `type` 

This enables hands-free text entry into any focused application. 

1 

## **Screenshot Capture** 

- Take screenshots instantly using voice commands 

## **Graphical User Interface (GUI)** 

Chronos includes a full configuration interface for managing settings and commands. 

The GUI can be accessed by saying: 

- `configuration` 

- `configure` 

- `settings` 

It can also be opened directly from the system tray. 

## **System Tray Integration** 

Chronos runs conveniently in the background with tray support for: 

- quick access to settings 

- fast application controls 

- easy exit and reload functionality 

## **User Experience** 

Chronos is designed with: 

- low-latency voice interaction • lightweight background performance • customizable command workflows • productivity-focused automation
